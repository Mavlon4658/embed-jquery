Занятие 1. Ввод и вывод данных

Задача «Самые последние часы»

Уровень сложности: Лёгкий

nik@developer

Условие

Напишите функцию, которая принимает 4 цифры и возвращает последнее время суток, которое может быть построено с использованием этих цифр. Время должно быть в формате ЧЧ:MM.

Пример

digits: 1, 9, 8, 3 => result: "19:38" digits: 9, 1, 2, 5 => result: "21:59" (19:25 is also a valid time, but 21:59 is later)

Примечания

  • Результатом должно быть действительное 24-часовое время, между 00:00 и 23:59.
  • Тестируются только те входные данные, которые имеют допустимые ответы.
